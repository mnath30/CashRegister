# CashRegister

CashRegister is a web based application which will tell the user the change to be returned to the customer using minimum number of currency notes.

## Features

The application takes the bill amount of the customer and the amount paid by the customer. It then calculates the balance and tells the user the number of notes of each denomination (2000, 500, 100, 20, 10, 5, 1) required to return the balance amount. 

It also checks for the following validations:
 * Bill amount and paid amount should not be zero
 * Bill amount and paid amount should not be negative
 * Bill amount should not be greater than paid amount

## Technology used

CashRegister is made using HTML, CSS, JavaScript. It is hosted on Netlify.
